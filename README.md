# AI-QUIZ-GENERATOR
Smart AI quiz generator app!!
🎯 AI Quiz Generator Web App
<br>
A smart and interactive AI-powered quiz generator that creates quizzes instantly based on selected topics and languages. Built using HTML, CSS, and JavaScript, this project provides a fun and engaging way to test knowledge.
<br>
<br>
🚀 Features
<br>
<br>
🤖 AI-generated quiz questions
<br>
📚 Topic selection (Technology, Science, History, etc.)
<br>
⏱️ Timer-based quiz system
<br>
✅ Instant feedback (green for correct, red for wrong)
<br>
🎨 Aesthetic animated UI with a bluish theme
<br>
📊 Score tracking
<br>
<br>
🛠️ Technologies Used
<br>
<br>
HTML5
<br>
CSS3 (Animated UI + Styling)
<br>
JavaScript (Logic & Functionality)
<br>
Gemini API (for AI question generation)
